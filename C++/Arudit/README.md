README

Этот код представляет собой пример программы для поиска слов на квадратной буквенной сетке.

Использование:

1. Введите буквенную сетку размером N x N.
2. Введите количество слов для поиска.
3. Для каждого слова, которое нужно найти:
   - Введите слово.
   - Программа автоматически начнет поиск этого слова на буквенной сетке.
   - Результаты поиска будут выведены на экран.

Примечание:

- Программа использует рекурсивный алгоритм для поиска слова на буквенной сетке.
- Поиск осуществляется по горизонтали, вертикали и диагонали.
- Результатом поиска будет сообщение о том, было ли найдено слово на сетке.
