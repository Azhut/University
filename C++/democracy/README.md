Программа определяет минимальное количество избирателей, которые могут изменить исход выборов.

Как это работает:
1. Сначала программа считывает количество групп избирателей и количество избирателей в каждой группе.
2. Затем программа сортирует группы избирателей по количеству избирателей в порядке возрастания.
3. Далее программа определяет минимальное количество групп, которые покрывают более половины всех избирателей.
4. Наконец, программа считает общее количество избирателей в минимальном числе групп и выводит результат.

Примечания:
- Программа предполагает, что каждая группа имеет нечетное количество избирателей.
