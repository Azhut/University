Программа для быстрого возведения матрицы в заданную степень.

Описание:
Программа реализует алгоритм быстрого возведения матрицы в степень, используя метод "разделяй и властвуй". Включает в себя функции для быстрого возведения числа в степень (binpow и FastPow), а также функцию для умножения двух матриц (MatrixMult).

Использование:
1. Определите исходную матрицу, представленную в виде двумерного массива.
2. Укажите желаемую степень, в которую нужно возвести матрицу.
3. Запустите программу, которая автоматически выполнит быстрое возведение матрицы в указанную степень.
4. Результат будет выведен на экран.

Примечания:
- Предполагается, что входная матрица квадратная.
- В программе реализован алгоритм быстрого возведения в степень, что позволяет ускорить процесс вычислений, особенно для больших степеней.
