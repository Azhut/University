README

Этот код представляет собой пример использования библиотеки SFML для создания анимированного персонажа в окне графического приложения.

Использование:

1. Задайте параметры анимации, используя методы класса Person:
   - `AddAnim`: добавляет новую анимацию персонажа, указывая путь к файлам кадров, имя анимации и количество кадров.
   - `SetDefaultSprite`: устанавливает спрайт по умолчанию для персонажа.
   - `Move`: перемещает персонажа в указанную позицию.
   - `SetScale`: устанавливает масштаб для персонажа.
   - `SetPosition`: устанавливает позицию персонажа.
   - `Animate`: анимирует персонажа по заданной анимации и времени между кадрами.

2. Запускайте главный цикл приложения, в котором вызывается метод `Animate` для анимации персонажа в окне приложения.

Примечание:

- Данный код создает окно приложения с использованием библиотеки SFML и отображает в нем анимированный персонаж.
- Анимации персонажа загружаются из файлов изображений, заданных путями к файлам.
- В коде также присутствует функционал для работы с текстурами и спрайтами персонажа.
