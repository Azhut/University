README

Этот код представляет собой набросок реализации класса Robot, который представляет собой робота с возможностью обучения и выполнения различных команд.

Использование:

1. Создайте экземпляр класса Robot.
2. Вызывайте различные методы класса для обучения робота, отката на предыдущий шаг, повторного обучения, создания копии робота или проверки его текущего состояния.

Примечание:

- Класс Robot имеет методы learn(), rollback(), relearn(), clone(), check(), которые позволяют обучать, откатывать, повторно обучать, создавать копии и проверять состояние робота соответственно.
- В данной реализации не представлены определения методов класса Robot, только их заголовки.
- Текущая реализация функции main() лишь иллюстрирует пример использования методов класса Robot через функцию DoCommand(), которая обрабатывает ввод пользовательских команд.
