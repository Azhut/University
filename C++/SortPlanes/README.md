# Конвертер координат и расстояний в трехмерном пространстве

## Описание
Программа позволяет работать с точками, прямыми и плоскостями в трехмерном пространстве. Основные функции включают расчет расстояний между точками и между точками и прямыми/плоскостями, а также сортировку плоскостей по расстоянию до заданной точки методом быстрой сортировки.

## Функционал
1. Возможность ввода точек, прямых и плоскостей.
2. Расчет расстояний между точками и между точками и прямыми/плоскостями.
3. Сортировка плоскостей по расстоянию до заданной точки методом быстрой сортировки.

## Компоненты
- **Vector**: структура для работы с векторами, включая основные операции и расчеты.
- **Line**: структура для работы с прямыми в пространстве, включая операции и расчеты расстояний.
- **Plane**: структура для работы с плоскостями в пространстве, включая операции и расчеты расстояний.
- **QuickSort**: функция для быстрой сортировки массива плоскостей по расстоянию до заданной точки.

## Использование
1. Введите количество плоскостей.
2. Введите координаты каждой плоскости.
3. Введите координаты заданной точки.
4. Программа выведет информацию о каждой плоскости и расстояние от нее до заданной точки.
5. Плоскости будут отсортированы по расстоянию до точки, и их информация будет выведена в отсортированном порядке.
