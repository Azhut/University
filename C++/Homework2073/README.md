Данная программа предназначена для отображения результатов соревнований по программированию в виде таблицы.

Использование:
1. Запустите программу.
2. Введите количество соревнований (NumberOfCompitition).
3. Для каждого соревнования введите следующие данные:
   - Название соревнования (CompName).
   - Дату проведения соревнования (Data).
   - Количество задач (NumberOfProblems).
   - Количество попыток на каждую задачу (NumberOfTries).
   - Для каждой попытки введите символ задачи (Task) и результат (result).
4. После ввода данных программа выведет таблицу с результатами каждого соревнования.

Примечания:
- Программа ожидает ввода данных в указанном выше формате.
- Результаты каждой попытки задачи представлены символами: 'o' (принято), 'x' (неправильное решение), '.' (попытка не предпринята).
- Для корректного вывода таблицы рекомендуется использовать ширину консоли не менее 80 символов.
- После завершения работы с программой закройте ее окно или нажмите клавишу "Ctrl+C" в командной строке.
