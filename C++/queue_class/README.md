# Быстрая сортировка для односвязного списка

Эта программа реализует алгоритм быстрой сортировки для упорядочивания односвязного списка по возрастанию.

## Функционал

- `push()`: Вставляет новый узел с заданными данными в начало связанного списка.
- `printList()`: Выводит элементы связанного списка.
- `getTail()`: Возвращает последний узел связанного списка.
- `partition()`: Проводит разделение связанного списка на основе опорного элемента.
- `quickSortRecur()`: Рекурсивно сортирует связанный список, используя алгоритм быстрой сортировки.
- `quickSort()`: Обертка для запуска быстрой сортировки на связанном списке.

## Структура

Программа определяет структуру `Node`, представляющую узел в связанном списке. Каждый узел имеет целочисленные данные и указатель на следующий узел.

## Использование

1. Скомпилируйте программу с помощью компилятора C++.
2. Запустите исполняемый файл.
3. Программа создаст связанный список с некоторыми элементами, отсортирует его с помощью быстрой сортировки и выведет отсортированный связанный список.

## Примечание

- Функция `main` демонстрирует использование алгоритма быстрой сортировки на связанном списке с примерными элементами.
- Алгоритм сортировки работает на месте без выделения дополнительной памяти.
