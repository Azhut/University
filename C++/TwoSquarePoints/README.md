# Создание квадратов

Программа создает квадраты на основе заданных координат вершин.

## Использование

Программа принимает количество квадратов `n` и координаты вершин каждого квадрата.

## Результат

Для каждого квадрата выводятся координаты всех его вершин.

## Ошибка

В функции `FillSq` передается копия структуры `Square`, поэтому изменения внутри функции не влияют на исходную структуру. Чтобы избежать этой проблемы, следует передавать структуру по ссылке или указателю.

## Примечание

Программа также повторно вычисляет координаты вершин квадратов после ввода.
