# Программа для работы с длинными строками как с числами

Эта программа на C++ позволяет складывать и вычитать длинные строки, представленные как числа, и записывать результаты в файл.

## Использование

1. Подготовьте файл `inp.txt` с двумя строками чисел для операций.
2. Запустите программу.
3. Результаты будут записаны в файл `out.txt` и выведены на экран.

## Формат входных данных

Файл `inp.txt` содержит две строки чисел без пробелов.

## Формат выходных данных

Результат операций записывается в файл `out.txt` в формате `a+b=result` для сложения и `a-b=result` для вычитания.

## Пример использования

### Входные данные (`inp.txt`):

123456789
987654321


### Выходные данные (`out.txt`):

123456789+987654321=1111111110
123456789-987654321=-864197532
