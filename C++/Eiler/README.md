Программа находит самую длинную цепь чисел, используя функцию Эйлера.

Как это работает:
- Считывает количество чисел (SIZE) и последовательно считывает числа.
- Для каждого числа вычисляет значение функции Эйлера (?).
- Если текущее значение ? меньше предыдущего, программа продолжает проверку.
- Обновляет значения максимальной длины цепи и позиции начала цепи при необходимости.
- Выводит начало и длину самой длинной цепи чисел.
