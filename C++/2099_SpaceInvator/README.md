README

Этот код представляет собой программу, которая выполняет проверку на валидность четырех точек в трехмерном пространстве, заданных векторами A, B, C и D. Дополнительно представлен конспект лекции Вахрушева, который содержит условия решения задачи.

Использование:

1. Введите координаты четырех точек A, B, C и D в трехмерном пространстве.
2. Программа выполнит проверку на валидность данных точек согласно условиям решения задачи.

Условия решения задачи (конспект лекции):

- AB должен быть перпендикулярен CD.
- AB и CD должны быть компланарны, что означает, что их векторное произведение должно равняться нулю: (AB * CD) = 0.
- Сначала должна быть указана точка A, затем B, C и D.
- Проверка на то, что точка C находится правее точки D.

Примечание:

- Класс Vector определяет вектор в трехмерном пространстве и предоставляет методы для выполнения различных операций над векторами.
- Класс Plane определяет плоскость в трехмерном пространстве и используется для дальнейшей проверки условий решения задачи.
- Программа выводит "Valid", если введенные точки образуют корректную конфигурацию, и "Invalid" в противном случае.
