README

Этот код представляет собой программу, которая разбирает список студентов из различных фракций Хогвартса (Slytherin, Hufflepuff, Gryffindor, Ravenclaw) и выводит список студентов для каждой фракции.

Использование:

1. Введите количество студентов (n).
2. Затем введите информацию о каждом студенте: имя студента, затем фракция (Slytherin, Hufflepuff, Gryffindor, Ravenclaw).

Примечание:

- Если введена неправильная фракция, программа выведет сообщение "your fucking idiot".
- Студенты разбираются и выводятся по фракциям в алфавитном порядке.
- Если для какой-либо фракции не указаны студенты, она будет пропущена в выводе.
