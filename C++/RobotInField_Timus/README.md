# Описание программы решения логических выражений

Эта программа предоставляет функционал для обработки и вычисления логических выражений.

## Функционал

- Удаление пробелов из выражения.
- Распознавание и замена следующих наборов символов:
  - `TRUE` -> `1`
  - `FALSE` -> `0`
  - `AND` -> `&`
  - `OR` -> `|`
  - `NOT` -> `!`
- Преобразование инфиксного выражения в постфиксное.
- Вычисление значения постфиксного логического выражения.
- Вычисление значения инфиксного логического выражения.

## Ввод

Программа не предоставляет собственный ввод. Входные данные должны быть предоставлены через интерфейс программы.

## Вывод

Результаты обработки и вычислений выводятся на экран.

## Примечания

- Программа предполагает, что логические выражения вводятся в виде инфиксных строк, например: `A & B | !C`.
- Программа поддерживает следующие логические операции: `AND`, `OR`, `NOT`.
- Программа преобразует входные данные таким образом, чтобы можно было вычислить их логическое значение.
- Входные данные должны содержать только логические операнды (`TRUE`, `FALSE`) и операторы (`AND`, `OR`, `NOT`).
