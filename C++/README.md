## Функциональное и логическое программирование

В этой папке вы найдете различные проекты, созданные в процессе учебных занятий по изучению языков программирования C и C++.

Основной акцент сделан на алгоритмы и структуры данных. 
## Структура проектов

Каждый проект находится в отдельной папке и включает в себя файлы исходного кода на языке C++, а также другие необходимые ресурсы и компоненты. Внутри каждой папки вы найдете:

- **Исходный код**: Файлы программы, содержащие функции, структуры данных, классы и другие элементы программирования.
- **Ресурсы**: Дополнительные файлы, такие как заголовочные файлы, текстовые файлы, изображения и т.д.
- **Документация**: Инструкции по использованию программы, описание функциональности и другие сведения.
