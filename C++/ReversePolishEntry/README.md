# Программа для конвертации и преобразования инфиксного выражения в постфиксное

Эта программа считывает строку, содержащую инфиксное выражение, и конвертирует его в постфиксное выражение, используя стек.

## Функционал

- Считывает строку с инфиксным выражением.
- Преобразует инфиксное выражение в постфиксное.
- Выводит конвертированное выражение.

## Ввод

Строка с инфиксным выражением.

## Вывод

Постфиксное выражение, полученное после конвертации.

## Описание алгоритма

1. Считывается строка с инфиксным выражением.
2. Преобразование инфиксного выражения в постфиксное выполняется в два этапа:
   - Сначала происходит конвертация инфиксного выражения в простой вид, поддерживаемый программой.
   - Затем происходит преобразование полученного простого выражения в постфиксное, используя стек.
3. Полученное постфиксное выражение выводится на экран.

## Замечания

- Программа поддерживает операции `+`, `-`, `*`, `/`, `^`, `(`, `)`, `|`, `&`, `!`.
- Операции `|`, `&`, `!` используются для логических выражений.
- Программа предполагает, что операнды и операции в инфиксном выражении разделены пробелами.
- Результат конвертации выводится на экран.
