# Программа для разбиения текста на строки с заданной шириной

Эта программа считывает текст из файла, разбивает его на строки с заданной шириной и записывает результат в другой файл.

## Функционал

- Считывает текст из входного файла.
- Разбивает текст на строки с заданной шириной.
- Записывает разбитый текст в выходной файл.

## Ввод

1. Ширина строки.
2. Текст из входного файла.

## Вывод

Текст, разбитый на строки с заданной шириной, записанный в выходной файл.

## Описание алгоритма

1. Считывается ширина строки и текст из входного файла.
2. Текст разбивается на строки с заданной шириной.
3. Разбитый текст записывается в выходной файл.

## Замечания

- Программа предполагает, что входной файл `inp.txt` содержит текст, а выходной файл `out.txt` будет использован для записи разбитого текста.
- Если строка текста превышает заданную ширину, она разбивается на несколько строк.
- Последняя строка записывается в файл независимо от ее ширины.
- Программа обрабатывает пробелы в тексте, чтобы распределить дополнительные пробелы равномерно между словами.
