Программа для работы с матрицей смежности

Функциональность:
- Чтение матрицы смежности из файла
- Вывод списка смежности для каждой вершины

Использование:
1. Создайте файл "Matrix.txt", содержащий матрицу смежности.
2. Запустите программу.

Функции:
- `PrintList(list<int> list)`: выводит элементы списка в обратном порядке.
- `main()`: основная функция программы, читает матрицу смежности, выводит списки смежности для каждой вершины.
