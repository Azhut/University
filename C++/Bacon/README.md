Поиск палиндромных слов в строках

Эта программа считывает строки из файла "inp.txt", находит в каждой строке наибольшее количество палиндромных слов и записывает их в файл "out.txt".

Как это работает:
1. Программа открывает файл "inp.txt" для чтения и файл "out.txt" для записи.
2. Для каждой строки в файле "inp.txt" программа:
    - Разбивает строку на слова.
    - Проверяет все возможные комбинации слов на палиндромность.
    - Выбирает комбинацию с максимальным количеством символов и записывает эти слова в файл "out.txt".

Примечания:
- Программа предполагает, что каждое слово разделено пробелом.
- В случае, если не удается открыть файлы "inp.txt" или "out.txt", программа выводит сообщение об ошибке.
