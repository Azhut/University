# Краткое описание программы

Эта программа считывает данные из файла, представляющего коэффициенты и степени многочлена, сортирует их по убыванию степеней, а затем вычисляет значение многочлена для заданного значения переменной x.

## Использование

1. Убедитесь, что у вас установлена среда выполнения C++.
2. Скомпилируйте программу с помощью компилятора C++.
3. Поместите входные данные в файл `in.txt`, указанный в коде.
4. Запустите программу.
5. Результаты будут выведены в консоль.

## Основные функции

- `Swap2(int i, int j, int *arr, int* arr2)`: Функция для обмена элементов массива.
- `IDK_sort(int*&arr, int start, int end, int*&arr2)`: Функция для сортировки массивов коэффициентов и степеней многочлена.
- `main()`: Основная функция программы. Считывает данные из файла, сортирует массивы, вычисляет значение многочлена и выводит результаты.

## Файлы

- `in.txt`: Входной файл, содержащий коэффициенты и степени многочлена.
- `out.txt`: Выходной файл (не используется).
