Программа для работы с матрицами

Функциональность:
- Ввод и вывод матрицы с форматированием
- Транспонирование матрицы

Использование:
1. Введите количество элементов и столбцов матрицы.
2. Введите элементы матрицы.
3. Результаты транспонирования и вывода матрицы будут выведены на экран.

Функции:
- `cinMatrix`: ввод матрицы из стандартного ввода.
- `coutMatrix`: вывод матрицы с форматированием.
- `DeleteMatrix`: освобождение памяти, занятой матрицей.
- `GetTransposedMatrix`: транспонирование матрицы.
- `StringToLen`: форматированное преобразование строки заданной длины.
- `GetElement`: получение элемента матрицы в строковом формате.
