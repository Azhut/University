# Краткое описание программы

Программа сортирует массив целых чисел с использованием алгоритма быстрой сортировки (quicksort) и выводит отсортированный массив.

## Использование

1. Убедитесь, что у вас установлена среда выполнения C++.
2. Скомпилируйте программу с помощью компилятора C++.
3. Запустите программу.
4. Результаты сортировки будут выведены в консоль.

## Основные функции

- `main()`: Основная функция программы. Создает массив и вызывает функцию сортировки.
- `qsortRecursive(int begin, int end, int *arr)`: Рекурсивная функция быстрой сортировки, сортирует часть массива.
- `qsort(int begin, int end, int* arr)`: Функция быстрой сортировки, но не используется в данной реализации.
