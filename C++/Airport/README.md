README

Этот код решает задачу, связанную с анализом последовательности строк, представляющих языки программирования. Поставленная задача заключается в определении наибольшего возможного числа строк, которые можно сгруппировать так, чтобы каждая группа содержала строки на разных языках программирования.

Использование:

1. Введите количество строк N, за которым следует список строк, представляющих языки программирования.
2. Программа вычислит наибольшее возможное число групп строк, удовлетворяющих условию задачи, и выведет результат.

Примечание:

- Программа выводит количество строк в каждой группе, разделенные пробелом.
- В случае, если задача не имеет решения, программа выведет "Igor is wrong."
