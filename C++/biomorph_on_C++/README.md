Программа рисует биоморфы, используя OpenGL.

Каждый биоморф представляет собой графическое представление комплексной функции, возводящейся в куб и изменяющей цвет в зависимости от результата.

Программа использует случайные значения цвета для закрашивания попавших в область биоморфов точек, а точки за пределами области закрашивает цветом фона.

Используются библиотеки: OpenGL и GLUT.
