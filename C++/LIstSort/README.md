# Односвязный список на C++

## Обзор

Предоставленный код реализует односвязный список на C++. Вот краткое описание его функциональности.

## Класс `List` (List.h)

- **Конструктор**: Инициализирует пустой список с размером 0 и указателем на начало списка, указывающим на нуль.
- **Деструктор**: Очищает список, удаляя все элементы.
- **push_front**: Вставляет новый элемент в начало списка.
- **push_back**: Вставляет новый элемент в конец списка.
- **insert**: Вставляет новый элемент на указанную позицию в списке.
- **pop_front**: Удаляет первый элемент списка.
- **pop_back**: Удаляет последний элемент списка.
- **removeAt**: Удаляет элемент на указанной позиции в списке.
- **clear**: Очищает список, удаляя все элементы.
- **Поддержка оператора []**: Позволяет обращаться к элементам списка по индексу.
- **GetSize**: Возвращает размер списка.

## Структура `Node` (List.h)

- Представляет собой узел в односвязном списке, содержащий данные типа `T` и указатель на следующий узел.

## Основная функция

- Создает экземпляр класса `List<int>`.
- Вставляет элементы в начало списка с помощью `push_front`.
- Выводит элементы списка с помощью перегруженного оператора `[]`.
- Удаляет последний элемент списка с помощью `pop_back`.
- Выводит обновленный список.

Основная функция демонстрирует базовые операции, такие как вставка и удаление элементов из списка.
