Данная программа считывает данные из текстового файла, содержащего информацию о нескольких матрицах, а именно их размерность и элементы. Затем программа обрабатывает эту информацию, находит минимальные значения диагональных элементов каждой треугольной матрицы и сортирует матрицы по возрастанию этих минимальных значений.

Использование:
1. Укажите путь к файлу с данными в переменной `filenameIn`.
2. Запустите программу.
3. Программа выведет на экран информацию о минимальных значениях диагональных элементов и элементы самих матриц, отсортированные по возрастанию минимальных значений.

Примечания:
- Программа ожидает, что файл с данными имеет следующую структуру:
  - Первая строка: количество матриц и их размерность (два целых числа, разделенные пробелом).
  - Далее идут данные о матрицах. Каждая матрица описывается несколькими строками, каждая из которых содержит элементы матрицы (целые числа), разделенные пробелами.
- Если не найдено ни одной треугольной матрицы, программа выведет сообщение "Here is no any triangular matrix".
