Данная программа предназначена для обработки текстового файла и проверки слов на палиндромность.

Использование:
1. Укажите путь к файлу, который содержит строки для обработки, в переменной `filename`.
2. Запустите программу.
3. Программа считает количество слов в каждой строке файла и выведет их на экран.
4. Затем программа будет проверять все возможные комбинации слов в строке на палиндромность и выводить результаты на экран.

Примечания:
- Программа ожидает, что каждая строка файла содержит слова, разделенные пробелами.
- В результате проверки каждой строки будет выведено сообщение о том, является ли комбинация слов палиндромом или нет.
- После завершения работы с программой закройте ее окно или нажмите клавишу "Ctrl+C" в командной строке.
