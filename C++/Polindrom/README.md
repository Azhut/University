# Краткое описание программы

Эта программа читает строки из входного файла, находит палиндромы в каждой строке и записывает результаты в выходной файл.

## Использование

1. Убедитесь, что у вас установлена среда выполнения C++.
2. Скомпилируйте программу с помощью компилятора C++.
3. Поместите входные данные в файл `inp.txt`, указанный в коде.
4. Запустите программу.
5. Результаты будут записаны в файл `out.txt`, указанный в коде.

## Основные функции

- `main()`: Основная функция программы. Считывает строки из файла, находит палиндромы и записывает результаты в файл.
- `StringToLen()`: Функция для форматированного преобразования строки заданной длины.
- `GetElement()`: Функция для получения элемента матрицы в строковом формате.
- `coutMatrix()`: Функция для вывода матрицы с форматированием.
- `DeleteMatrix()`: Функция для освобождения памяти, занятой матрицей.
- `GetTransposedMatrix()`: Функция для транспонирования матрицы.

## Файлы

- `inp.txt`: Входной файл, из которого считываются строки.
- `out.txt`: Выходной файл, в который записываются результаты.
