### README: Генераторы в Python

---

### Описание:

Этот репозиторий содержит несколько скриптов на Python, демонстрирующих использование генераторов в Python.

### Скрипты:

1. **generate_dates.py**: Этот скрипт содержит генераторную функцию `generate_dates`, которая выдает последовательные даты, начиная с заданной даты. Он принимает месяц, день и год в качестве входных данных и бесконечно генерирует даты.

2. **generate_dow.py**: Этот скрипт содержит генераторную функцию `generate_dow`, которая выдает последовательные дни недели (понедельник, вторник и т. д.), начиная с заданной даты. Он принимает месяц, день и год в качестве входных данных и бесконечно генерирует дни недели.

3. **generate_mixed_date.py**: Этот скрипт объединяет функциональность двух предыдущих скриптов. Он генерирует даты вместе с соответствующими днями недели.

4. **head.py**: Этот скрипт реализует генераторную функцию `head`, которая считывает первые `num_lines` строк из указанного файла и выдает их по одной. Он принимает путь к файлу и количество строк для чтения в качестве входных данных.

5. **prime_generator.py**: Этот скрипт содержит генераторную функцию `prime_generator`, которая выдает простые числа бесконечно. Он начинается с 2 и генерирует простые числа одно за другим по мере запроса.



### Примечание:

- Генераторы эффективны по памяти, так как они производят значения по мере необходимости, что делает их подходящими для обработки больших наборов данных или бесконечных последовательностей.
- Каждый скрипт может быть запущен независимо, чтобы наблюдать поведение соответствующей генераторной функции.
