# Описание

Этот код реализует функции `base64_encode` и `base64_decode` для кодирования и декодирования текста с использованием алгоритма Base64. Base64 - это способ представления бинарных данных в текстовом формате, который состоит из 64 различных ASCII символов.

## Функция base64_encode

Функция `base64_encode` кодирует входной текст с использованием алгоритма Base64.

### Алгоритм

1. Входной текст переводится в байтовый формат с помощью кодировки UTF-8.
2. Если длина текста не кратна 3, то добавляются нулевые байты, чтобы длина стала кратной 3.
3. Текст разбивается на блоки по 3 байта.
4. Каждый блок байтов преобразуется в число.
5. Каждое число разбивается на 4 группы по 6 бит.
6. Значение каждой группы используется как индекс в таблице Base64 для получения символа.
7. Полученные символы объединяются в строку.

## Функция base64_decode

Функция `base64_decode` декодирует закодированный текст, восстанавливая исходный текст.

### Алгоритм

1. Каждый символ из закодированного текста используется как индекс в таблице Base64, чтобы получить значение в бинарном формате.
2. Бинарные значения каждого символа объединяются в строку.
3. Каждые 8 бит преобразуются обратно в символ.
4. Полученные символы объединяются в строку.

## Примечание

- В коде также реализовано добавление символов "=" в конце закодированного текста для обеспечения правильного количества битов при декодировании.
- Данный код предоставляет демонстрацию использования алгоритма Base64 и может быть использован в различных проектах для кодирования и декодирования данных.
