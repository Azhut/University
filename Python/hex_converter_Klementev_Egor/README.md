# Описание

Программа `hex_converter_Klementev_Egor` представляет собой функцию `hex_dump`, которая выполняет шестнадцатеричное представление данных в формате дампа памяти.

## Функциональность

Функция `hex_dump` принимает входные данные и выводит их в формате дампа памяти, где каждый байт представлен в шестнадцатеричном формате, а также символьное представление, если байт соответствует символу в диапазоне от 32 до 126. В противном случае используется символ точки (`.`).

## Пример использования

```python
data = b'\x48\x65\x6c\x6c\x6f\x20\x57\x6f\x72\x6c\x64\x48\x65\x6c\x6c\x6f\x20\x57\x6f\x72\x6c\x64'
hex_dump(data)
```

## Вывод

```
00000000: 48 65 6c 6c 6f 20 57 6f 72 6c 64 48 65 6c 6c 6f  Hello WorldHello
00000010: 20 57 6f 72 6c 64                                  World
```

Каждая строка вывода состоит из:
- Смещение (offset) в шестнадцатеричном формате.
- Шестнадцатеричное представление данных.
- Символьное представление данных.
