### Python Decorators README

---

### Описание:

Этот репозиторий содержит примеры использования декораторов в Python.

### Файлы:

1. **reverse_string_decorator.py**: Пример использования декоратора, который переворачивает строку, возвращаемую функцией, если это возможно.

2. **prime_filter_decorator.py**: Пример использования декоратора, который фильтрует список целых чисел, оставляя только простые числа.

3. **simple_decorator.py**: Пример создания "хорошего" декоратора, который сохраняет магические поля `__name__` и `__doc__` функции, к которой он применяется.

### Использование:

- Каждый файл содержит примеры использования декораторов в Python.
- Для запуска примеров, используйте команду `python filename.py`.
