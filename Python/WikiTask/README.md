
README

Этот скрипт предназначен для извлечения ссылок на странице Википедии.

1. Пользователь указывает URL-адрес страницы Википедии.
2. Скрипт отправляет GET-запрос к указанному URL-адресу и получает HTML-код страницы.
3. Используя библиотеку lxml, скрипт анализирует HTML-код и извлекает все ссылки, содержащие домен "wikipedia.org".
4. Полученные ссылки выводятся на экран.

Для работы скрипта необходимы следующие библиотеки:
- httpx (для отправки HTTP-запросов)
- lxml (для обработки HTML-кода)

Этот скрипт может быть полезен для извлечения информации с веб-страницы Википедии для последующего анализа или обработки.
