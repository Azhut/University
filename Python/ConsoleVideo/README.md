## Описание

Этот скрипт преобразует видеофайл в ASCII-ART, используя яркость пикселей для определения соответствующего символа из заданного набора. Процесс обработки происходит кадр за кадром, преобразуя каждый кадр в ASCII-ART изображение и выводя его в консоль.

### Библиотеки

- **cv2 (OpenCV)**: Используется для загрузки видео и обработки каждого кадра.
- **time**: Используется для задержки между кадрами.

### Переменные

- **symbols**: Набор символов, используемых для отображения яркости пикселей.
- **video**: Объект видео, загруженный из файла 'BlackWhite.mp4'.
- **art_width**: Ширина ASCII-ART изображения (в символах).
- **resized_frame**: Кадр, масштабированный до размера ASCII-ART изображения.
- **ascii_image**: Строка, содержащая ASCII-ART представление кадра.

### Основной алгоритм

1. Каждый кадр видео масштабируется до заданной ширины ASCII-ART изображения.
2. Для каждого пикселя в масштабированном кадре определяется яркость.
3. Яркость пикселя используется для определения соответствующего символа из набора `symbols`.
4. Полученные символы объединяются в строки, формируя ASCII-ART изображение кадра.
5. ASCII-ART изображение кадра выводится в консоль.
6. Процесс повторяется для каждого кадра видео с небольшой задержкой между кадрами.

### Примечание

- Для запуска этого скрипта требуется установленная библиотека OpenCV.
- Качество ASCII-ART изображения зависит от размера исходного видео, а также от выбранного набора символов.
