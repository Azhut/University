### Translate Text README

---

### Описание:

Этот скрипт предназначен для автоматического перевода слов из текстового файла на английском языке на русский язык с использованием API Google Translate.

### Функциональность:

- Считывает текст из файла `Text.txt`.
- Подсчитывает количество слов и символов в тексте.
- Переводит каждое слово из текста с английского на русский.
- Пользователь может выбрать, записать ли переведенное слово в файл `Output.txt` или пропустить его.
- Записывает результаты в файл `Output.txt`.

### Зависимости:

- Python 3.x
- Библиотека requests для отправки запросов на сервер Google Translate.

### Использование:

1. Убедитесь, что у вас установлен Python 3.x.
2. Установите библиотеку requests, если она не установлена: `pip install requests`.
3. Поместите текст, который нужно перевести, в файл `Text.txt`.
4. Запустите скрипт `translate_text.py`.
5. Следуйте инструкциям в консоли для записи переведенных слов в файл или пропуска слов.
