### Flask Web App README

---

### Описание:

Это пример веб-приложения на Flask, которое отображает домашнюю страницу с некоторыми постами. В приложении используется шаблонизатор Jinja2 для создания динамического контента.

### Функциональность:

- Приложение отображает домашнюю страницу с постами пользователей.
- Посты представлены в виде списка с авторами и содержанием.
- Имя текущего пользователя отображается вверху страницы.
- Каждый пост может содержать текст на разных языках, включая русский.
