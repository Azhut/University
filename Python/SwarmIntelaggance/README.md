Этот скрипт на Python создает анимацию движения случайного роя частиц в двумерном пространстве с использованием библиотеки Matplotlib.

## Как использовать:

1. Убедитесь, что у вас установлен Python и библиотеки NumPy и Matplotlib.
2. Запустите скрипт.
3. Вы увидите график с анимацией движения роя частиц в прямоугольной области с координатами от 0 до 1 по обеим осям.

## Как это работает:

- Сначала создаются случайные координаты и скорости для частиц роя.
- Затем создается график с осями от 0 до 1.
- Функция `update_plot` обновляет положения частиц на каждом кадре и обновляет график с помощью метода `set_offsets`.
- Анимация создается с использованием `FuncAnimation`, который вызывает функцию `update_plot` для обновления графика на каждом кадре.

Вы можете изменить количество частиц, длительность анимации и другие параметры, чтобы получить желаемый результат.
