README

Этот скрипт предназначен для оценки пригодности студента для выбранной специальности.

1. Пользователь вводит номер студента.
2. Загружается информация о студентах из файла `students.json`.
3. Если студент с указанным номером существует, выводится его имя и навыки.
4. Пользователь выбирает специальность для оценки студента.
5. Загружается информация о специальностях из файла `professions.json`.
6. Если выбранная специальность существует, оценивается пригодность студента для этой специальности.
7. Выводится процент пригодности студента для выбранной специальности, а также навыки, которыми он обладает, и навыки, которые ему не хватает для успешного обучения по этой специальности.

Данные о студентах и специальностях загружаются из соответствующих файлов `students.json` и `professions.json` при помощи функций `load_students` и `load_professions`. В файле `utils.py` содержатся функции для загрузки данных, поиска студентов и профессий, а также оценки пригодности студента для выбранной специальности.
