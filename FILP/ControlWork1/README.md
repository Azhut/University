## Краткое описание:
Контрольная работа


Этот код представляет собой пример реализации односвязного списка на языке C++. Включает в себя класс `CList`, представляющий список, и некоторые его методы.

### Класс `CList`:

- `CItem`: Вложенная структура, представляющая элемент списка с данными типа `T`.
- `head`: Указатель на начало списка.
- `CList()`: Конструктор по умолчанию, инициализирует пустой список.
- `CList(T* t, int n)`: Конструктор, инициализирующий список элементами массива `t` длиной `n`.
- `~CList()`: Деструктор, освобождающий память, выделенную под элементы списка.
- `insFirst(T t)`: Вставляет элемент `t` в начало списка.
- `insertAfterEveryFourth(T value)`: Вставляет элемент `value` после каждого четвертого элемента списка.
- `copySet()`: Создает копию списка, исключая повторяющиеся элементы.
- `print()`: Выводит элементы списка на экран.
- `size()`: Возвращает количество элементов в списке.

### Основная логика:

1. Создаются два списка `L1` и `L2`, первый из которых пуст, а второй инициализируется массивом `a`.
2. Выполняются операции вставки в начало списка и вывода на экран.
3. Вызывается метод `insertAfterEveryFourth()` для второго списка `L2`, вставляющий значение после каждого четвертого элемента.
4. Создается копия списка `L2`, исключая повторяющиеся элементы.
5. Создается третий список `L3` и его копия `L3_SET`, исключая повторяющиеся элементы.
