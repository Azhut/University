## Краткое описание:

Этот код демонстрирует использование линейного аллокатора и его применение в контейнере `std::map`. Линейный аллокатор позволяет эффективно управлять памятью, выделяя и освобождая блоки памяти из единого большого блока.

### Основные компоненты:

- `LinearAllocator`: Класс, реализующий линейный аллокатор для управления памятью.
- `CStringComparator`: Функтор для сравнения строк в контейнере `std::map`.
- `TextMapTest()`: Функция, демонстрирующая использование линейного аллокатора в контейнере `std::map`.
- `GetCurrentTimeMs()`: Функция для получения текущего времени в миллисекундах.

### Основная логика:

1. Создается объект `LinearAllocator<void>`, который может выделять память произвольного типа.
2. Выполняются операции выделения и освобождения памяти с использованием линейного аллокатора.
3. Вызывается функция `TextMapTest()`, которая использует линейный аллокатор для контейнера `std::map`.
4. Измеряется время выполнения функции `TextMapTest()`.
