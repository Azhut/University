## Краткое описание:

Этот код считывает содержимое файла и подсчитывает количество вхождений каждого слова, используя контейнер `std::map`. Затем он сортирует слова по количеству их вхождений и выводит результаты на экран.

### Основные компоненты:

- `compare()`: Функция сравнения для сортировки слов по количеству вхождений.
- `sort()`: Функция для сортировки слов и вывода результатов на экран.
- `main()`: Основная функция, в которой выполняется считывание файла, подсчет слов и их сортировка.

### Основная логика:

1. Открывается файл для чтения.
2. Считывается содержимое файла, слово за словом, и подсчитывается количество вхождений каждого слова с использованием контейнера `std::map`.
3. После завершения чтения файла слова сортируются по количеству вхождений с помощью функции `sort()`.
4. Отсортированные слова и их количество выводятся на экран.
5. Измеряется время выполнения программы.

### Примечание:

- Вывод программы можно закомментировать для проверки времени выполнения без вывода на экран.
