## Краткое описание:
Контрольная работа по алгоритмам

Этот код выполняет поиск пути в лабиринте из файла и записывает результат в другой файл. Используется алгоритм поиска в ширину (BFS).

### Основные компоненты:

- `Point`: Структура, представляющая координаты точки в лабиринте.
- `isValid()`: Функция для проверки, является ли точка допустимой для перемещения.
- `findPath()`: Функция для поиска пути в лабиринте с использованием алгоритма BFS.
- `main()`: Основная функция, считывающая лабиринт из файла, выполняющая поиск пути и записывающая результат в другой файл.

### Основная логика:

1. Открывается файл для чтения входных данных (`in.txt`) и файл для записи результатов (`out.txt`).
2. Считывается размер лабиринта (количество строк и столбцов) и сам лабиринт из файла.
3. Затем считываются координаты начальной и конечной точек.
4. Вызывается функция `findPath()`, которая находит путь в лабиринте с использованием алгоритма BFS.
5. Если путь найден, координаты точек пути записываются в выходной файл, иначе записывается "N".
6. Файлы закрываются, завершается выполнение программы.

### Примечание:

- Путь выводится в файл в формате "Y" (если путь найден) или "N" (если путь не найден), а затем координаты точек пути (если путь найден).
- Файлы считываются и записываются из указанных в коде путей.
