## Краткое описание:

Данный код реализует простой тест, использующий стандартный контейнер `std::map`, кастомный компаратор строк `CStringComparator` и кастомный аллокатор `CMyAllocator`. Этот тест направлен на демонстрацию использования кастомного аллокатора в контейнере `std::map` для работы с динамически выделенными строками.

### Основные компоненты:

- `CStringComparator`: Класс компаратора для сравнения строк.
- `CMyAllocator`: Класс кастомного аллокатора.
- `TextMapTest()`: Функция, выполняющая тест с использованием `std::map`, `CStringComparator` и `CMyAllocator`.

### Основная логика:

1. Создается объект `std::map` с ключом `char*` и значением `size_t`, используя кастомный компаратор `CStringComparator` и кастомный аллокатор `CMyAllocator`.
2. В цикле добавляются строки в `std::map`. Если строка уже присутствует в карте, увеличивается счетчик ее встречаемости.
3. Выводится содержимое `std::map`, показывая количество встречаемости каждого слова.

### Примечание:

- Кастомный аллокатор `CMyAllocator` используется для управления памятью для строк в `std::map`. Это демонстрирует, как можно заменить стандартные операции выделения и освобождения памяти в контейнере.
- Кастомный компаратор `CStringComparator` определяет порядок сортировки строк в `std::map` на основе их лексикографического порядка.
- Код написан на C++, используются стандартные библиотеки и некоторые стандартные функции для работы с памятью (например, `malloc` и `free`).
