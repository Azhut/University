## Краткое описание:

Этот код демонстрирует реализацию простого односвязного списка с использованием шаблонов классов в C++.

### Основные компоненты:

- `CList`: Шаблонный класс, представляющий односвязный список.
- `CItem`: Вложенная структура, представляющая элемент списка.
- `InsFirst()`: Метод для вставки нового элемента в начало списка.
- `InsLast()`: Метод для вставки нового элемента в конец списка.
- `print()`: Метод для вывода списка на экран.
- `printMemoryState()`: Метод для отладочного вывода состояния памяти списка.
- `getHead()`: Метод для получения указателя на голову списка.

### Основная логика:

1. Создается список типа `int`, в который вставляются элементы сначала и с конца.
2. Создается список типа `std::string`, инициализируемый массивом строк, который также выводится на экран.
3. Для отладки используется метод `printMemoryState()`, который выводит состояние памяти списка.

### Примечание:

- Реализация предполагает простое использование списка с базовыми операциями вставки и вывода.
- Шаблон класса позволяет создавать списки с элементами различных типов.
