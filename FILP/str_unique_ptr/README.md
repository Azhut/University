## Краткое описание:

Этот код реализует класс `str`, представляющий строку символов без завершающего символа `'\0'`. Используется умный указатель `std::unique_ptr<char[]>` для управления памятью строки.

### Основные компоненты:

- `str`: Класс, представляющий строку символов.
- `s`: Умный указатель на массив символов.
- `n`: Размер строки символов.
- `str(const char *st)`: Конструктор для создания объекта `str` из строки `st`.
- `str(const str& T)`: Конструктор копирования.
- `operator char* ()`: Перегрузка оператора преобразования типа `str` в строку `char*`.
- `operator =`: Перегрузка оператора присваивания.
- `print()`: Метод для печати строки.
- `length()`: Метод для получения длины строки.
- `at()`: Метод для получения символа по индексу.
- `setchar()`: Метод для установки символа по индексу.

### Основная логика:

1. Создаются объекты `str` с использованием различных конструкторов.
2. Печатается приветственное сообщение на русском языке.
3. Продемонстрированы операции присваивания, доступа к символам строки и изменения символов.

### Примечание:

- Класс `str` предоставляет базовые операции для работы со строками, такие как создание, копирование, присваивание и доступ к символам.
- Используется умный указатель `std::unique_ptr<char[]>` для автоматического освобождения памяти, что помогает предотвратить утечки памяти.
- Для вывода строки на экран используется метод `print()`, а для получения длины строки - метод `length()`.
