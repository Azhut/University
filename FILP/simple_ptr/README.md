## Краткое описание:

Этот код реализует умный указатель (smart pointer) с подсчетом ссылок. Умный указатель автоматически управляет памятью, освобождая выделенную память, когда на объект больше нет ссылок.

### Основные компоненты:

- `SmartPtr`: Шаблонный класс умного указателя.
- `ptr`: Указатель на объект.
- `pCounter`: Указатель на счетчик ссылок.
- `SmartPtr(T *p = nullptr)`: Конструктор для создания умного указателя.
- `SmartPtr(const SmartPtr &other)`: Конструктор копирования.
- `~SmartPtr()`: Деструктор, освобождающий память при необходимости.
- `operator=`: Перегрузка оператора присваивания.
- `operator*()`: Перегрузка оператора разыменования.
- `operator->()`: Перегрузка оператора селектора выбора.
- `use_count()`: Метод для получения количества ссылок на объект.

### Основная логика:

1. Создаются объекты типа `SmartPtr<int>` и выделяется память для хранения значения `10`.
2. Создается второй объект `SmartPtr<int>`, который копирует первый объект. Оба объекта теперь указывают на одну область памяти.
3. Создается третий объект `SmartPtr<int>`, который указывает на новую область памяти.
4. Значение объекта `ptr3` затем присваивается значению объекта `ptr2`. После этого два объекта указывают на одну область памяти, а счетчик ссылок соответствующе увеличивается.

### Примечание:

- Умный указатель автоматически освобождает память, когда на объект больше нет ссылок, что предотвращает утечки памяти.
- Количество ссылок на объект можно получить с помощью метода `use_count()`.
- Умный указатель нельзя использовать с объектами, созданными с помощью `new[]`, так как он не имеет информации о размере выделенного блока памяти.
