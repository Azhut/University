## Краткое описание

Этот код реализует простой алгоритм кодирования и декодирования сообщений с использованием переменной длины кода. Программа запрашивает ввод от пользователя, кодирует введенное сообщение, а затем декодирует его обратно.

### Кодирование

1. Программа принимает ввод пользователя с помощью библиотеки `readline`.
2. Она анализирует введенное сообщение и строит словарь символов с их бинарными представлениями.
3. После построения словаря программа кодирует введенное сообщение, заменяя каждый символ его бинарным представлением.

### Декодирование

1. Пользователю предлагается ввести закодированное сообщение.
2. Программа декодирует сообщение, сопоставляя каждую последовательность битов с символом из словаря.
