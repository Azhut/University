## Краткое описание

Этот код реализует алгоритм построения префиксного кода Хаффмана для сжатия текстовых данных. Программа запрашивает текстовую строку у пользователя, строит дерево Хаффмана на основе частоты встречаемости символов в строке, а затем кодирует и декодирует текст, используя построенный код.

### Функции кодирования и декодирования

- `Incode(string, CodeOfChar)`: Кодирует входную строку, заменяя каждый символ его бинарным представлением, в соответствии с построенным префиксным кодом.
- `Decode(code, CharFromCode)`: Декодирует закодированную строку, заменяя каждую последовательность битов на соответствующий символ, используя словарь префиксного кода.

### Алгоритм построения дерева Хаффмана

1. Программа анализирует введенную строку и определяет частоту встречаемости каждого символа.
2. На основе частоты символов строится дерево Хаффмана.
3. Каждый лист дерева представляет символ и его код в префиксном коде.

### Ввод и вывод

- Пользователю предлагается ввести текстовую строку.
- Программа выводит код каждого символа, а также закодированную строку.
- Затем пользователю предлагается ввести закодированную строку.
- Программа декодирует строку и выводит результат.

### Примечание

Этот код предоставляет простую демонстрацию алгоритма Хаффмана и может быть доработан для обработки различных сценариев и оптимизации производительности.
