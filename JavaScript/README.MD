## Информационные Системы и технологии

В этой папке вы найдете различные проекты, созданные в процессе учебных занятий по изучению алгоритмов на примере языка программирования JavaScript.

## Структура проектов

Каждый проект находится в отдельной папке и включает в себя файлы исходного кода на языке JavaScript, а также другие необходимые ресурсы и компоненты.
