### Поиск подстроки в строке: скрипт на JavaScript

Этот скрипт реализует два алгоритма поиска подстроки в строке: Бойера-Мура-Хорспула и Бойера-Мура.

### Как использовать

1. **Подготовка данных:**
   - Создайте файл `input2.txt`.
   - В первой строке поместите исходную строку, а во второй - подстроку для поиска.
   - Обе строки разделяйте переносом строки.

2. **Запуск скрипта:**
   - Убедитесь, что у вас установлен Node.js.
   - Откройте терминал в каталоге с файлом скрипта и данными.
   - Выполните команду: `node filename.js`, где `filename.js` - имя скрипта.

### Вывод результатов

Результаты поиска выводятся в консоль.

### Детали алгоритмов

1. **Бойер-Мур-Хорспул:**
   - Использует предварительно вычисленную таблицу сдвигов.
   - Быстрый, особенно на случайных текстах.

2. **Бойер-Мур:**
   - Использует таблицу сдвигов и таблицу остановок.
   - Позволяет избежать проверок, если образец не соответствует тексту.

### Примечание

- Скрипт считывает данные из файла `input2.txt`.
- Результаты поиска выводятся в консоль в виде массива индексов всех вхождений подстроки в строку.
