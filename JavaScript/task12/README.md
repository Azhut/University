## Краткое описание

Этот код реализует алгоритм построения оптимального префиксного дерева Хаффмана для сжатия данных. Программа запрашивает текстовую строку у пользователя, вычисляет частоту встречаемости каждого символа, строит дерево Хаффмана на основе этой информации, кодирует и декодирует сообщения, используя построенное дерево.

### Основные шаги

1. **Считывание и подготовка данных**: Программа запрашивает у пользователя ввод текстовой строки и вычисляет частоту встречаемости каждого символа.

2. **Построение дерева Хаффмана**: На основе полученной информации программа строит префиксное дерево Хаффмана, используя рекурсивный алгоритм деления на поддеревья.

3. **Кодирование сообщения**: Программа кодирует введенную строку, заменяя каждый символ его бинарным представлением в соответствии с префиксным кодом Хаффмана.

4. **Декодирование сообщения**: Пользователю предлагается ввести закодированное сообщение. Программа декодирует его, восстанавливая исходное сообщение на основе построенного дерева.

### Примечание

- Программа выводит информацию о частоте каждого символа, построении дерева Хаффмана и коде каждого символа для дальнейшего использования.
- Данный код предоставляет демонстрацию алгоритма Хаффмана и может быть улучшен для обработки различных сценариев и оптимизации производительности.
