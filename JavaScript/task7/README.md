## Краткое описание

Этот код реализует два алгоритма поиска подстроки в строке: алгоритм Бойера-Мура-Хорспула и алгоритм Бойера-Мура. Оба алгоритма используются для эффективного поиска подстроки в строке.


## Использование

1. Создайте файл `input2.txt`, в котором первая строка содержит текст, в котором вы хотите найти подстроку, а вторая строка содержит саму подстроку.
2. Запустите код с помощью Node.js:


node <имя_файла>.js


## Результаты

После выполнения кода он выведет в консоль массивы индексов, где была найдена подстрока, с использованием обоих алгоритмов:

- **Boyer-Moore-Horsepool**: Индексы, где найдена подстрока с использованием алгоритма Бойера-Мура-Хорспула.
- **Boyer-Moore**: Индексы, где найдена подстрока с использованием алгоритма Бойера-Мура.
