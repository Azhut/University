### Краткое описание скрипта поиска подстроки в строке

Этот скрипт на языке JavaScript выполняет поиск всех вхождений подстроки в строку, используя различные алгоритмы: Z-функцию, Префикс-функцию, Алгоритм Кнута-Морриса-Пратта и Автомат.

### Как использовать

1. Создайте файл `input2.txt` и поместите в него две строки: исходную строку и подстроку для поиска.
2. Запустите скрипт с помощью Node.js, используя команду `node filename.js`, где `filename.js` - имя файла скрипта.

### Примечание

- Скрипт считывает данные из файла `input2.txt`.
- Результаты поиска выводятся в консоль.
- Поддерживается поиск всех вхождений подстроки в строку.
