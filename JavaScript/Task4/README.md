# Поиск подстроки в строке

Этот репозиторий содержит три скрипта на языке JavaScript для поиска подстроки в строке:
1. BruteForce.js - скрипт, реализующий алгоритм "грубой силы" для поиска подстроки в строке.
2. HashMaping.js - скрипт, использующий хэш-карты для поиска подстроки в строке.
3. RobinKartp.js - скрипт, реализующий алгоритм Робина-Карпа для поиска подстроки в строке.

## Использование

1. Запустите каждый скрипт, передав в качестве аргументов строку и подстроку для поиска.
2. Скрипты считывают данные из файла `input2.txt`, где первая строка содержит исходную строку, а вторая - подстроку для поиска.

## Описание скриптов

- BruteForce.js: Реализация алгоритма "грубой силы" для поиска подстроки в строке. Перебирает все возможные смещения подстроки в строке.
- HashMaping.js: Использует хэш-карты для быстрого поиска подстроки в строке. Вычисляет хэш-значение подстроки и сравнивает его с хэш-значениями подстрок строк.
- RobinKartp.js: Реализация алгоритма Робина-Карпа для поиска подстроки в строке. Использует хэширование и сравнение хэш-значений подстроки с хэш-значением строк.
