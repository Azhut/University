## Краткое описание

Этот код представляет собой несколько конструкторов JavaScript для моделирования группы студентов, меню и треугольника, а также функции для их обработки.

### Конструктор Gruppa

Конструктор `Gruppa` создает объект группы студентов с заданными характеристиками:

- `n`: номер группы,
- `spec`: специализация группы,
- `kolich`: количество студентов в группе.

Конструктор также содержит методы:

- `add_stud`: добавляет указанное количество студентов в группу и выводит информацию о добавлении,
- `sub_stud`: исключает указанное количество студентов из группы и выводит информацию об исключении.

### Объекты групп

Созданы три объекта группы: `gr2`, `gr3`, и `gr4`, каждый из которых представляет группу студентов с уникальными характеристиками. Произведены операции добавления и исключения студентов из групп, после чего выведено обновленное количество студентов в каждой группе.

### Объект меню

Создан объект `menu`, представляющий собой меню с заданными шириной, высотой и заголовком.

### Функции обработки объектов

1. `AddString(obj)`: Добавляет к каждому строковому значению в объекте строку " Это строка".
2. `Log_obj(obj)`: Выводит все свойства объекта в консоль.

### Конструктор Triangle

Конструктор `Triangle` создает объект треугольника с заданными сторонами (`side1`, `side2`, `side3`). Конструктор содержит методы:

- `Perimetr`: вычисляет периметр треугольника,
- `Square`: вычисляет площадь треугольника по формуле Герона,
- `Radius`: вычисляет радиус вписанной окружности в треугольник.

Для созданного объекта `Triangle1` выводятся значения периметра, площади и радиуса вписанной окружности.
