## Краткое описание

Этот код представляет собой программу, которая вычисляет энтропию строки, введенной пользователем.

### Чтение ввода

Программа использует библиотеку `readline` для чтения ввода пользователя с консоли. После ввода строки программа закрывает интерфейс чтения.

### Вычисление энтропии

Программа вычисляет энтропию строки, используя формулу Шеннона для дискретной случайной величины:

\[ H = - \sum_{i=1}^{n} p(x_i) \cdot \log_{n}(p(x_i)) \]

где:
- \( n \) - количество различных символов в строке,
- \( p(x_i) \) - вероятность появления символа \( x_i \) в строке.

### Расчет вероятностей

Программа вычисляет количество различных символов в строке и вероятность появления каждого символа. Затем она вычисляет энтропию строки, используя полученные вероятности.

### Вывод результата

После вычисления энтропии программа выводит ее значение в консоль.

### Примечание

Этот код предоставлен в качестве демонстрации вычисления энтропии строки и может быть доработан или использован в соответствии с вашими потребностями.
