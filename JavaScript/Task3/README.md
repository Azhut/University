# Кодирование текстового файла

Этот скрипт предназначен для кодирования текстового файла `file.txt` и записи результата в файл `code.txt`.

## Использование

1. Создайте два текстовых файла: `file.txt` и `code.txt`.
2. Укажите путь к этим файлам в переменной `path` в скрипте `encoder.js`.
3. Запустите скрипт из командной строки с помощью команды `Cscript <путь_к_файлу>\encoder.js`.

## Примечание

- Скрипт считывает содержимое файла `file.txt`.
- Каждый символ заменяется на его код с указанием количества повторений символа.
- Закодированный текст сохраняется в файл `code.txt`.

**Важно:** Для работы скрипта необходимо иметь активированный ActiveXObject, что может не поддерживаться на всех операционных системах и в средах выполнения.
