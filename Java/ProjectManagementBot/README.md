##Учебный проект

Телеграмм бот, в котором реализуется менеджмент проектов

### Класс `App`
- `main(String[] args)`: Точка входа в приложение. Инициализирует `TelegramView` и запускает диалог.

### Класс `TelegramView`
- `TelegramView`: Конструктор, инициализирующий HashMap `usersBase`.
- `getBotUsername()`: Возвращает имя бота.
- `getBotToken()`: Возвращает токен бота из переменных среды.
- `onUpdateReceived(Update update)`: Переопределяет метод для обработки обновлений от пользователей.
  - Обрабатывает текстовые сообщения от пользователей, извлекая сообщение, данные пользователя и создавая `Request`.
  - Управляет `usersBase`, сопоставляя имена пользователей с экземплярами `Controller`.
  - Обрабатывает различные типы ответов (`TEXT`, `DATE`) и отправляет соответствующие ответы пользователям.
  - Управляет взаимодействиями с календарем, такими как переключение между месяцами, выбор дат и отправка запросов календаря.
- `sendCalendarRequest(String chatId)`: Отправляет запрос календаря пользователю с указанным `chatId`.
- `deletePreviousCalendar(String chatId)`: Удаляет предыдущее сообщение с календарем, отправленное пользователю.
- `sendText(String userId, String text)`: Отправляет текстовое сообщение пользователю с указанным `userId`.
- `startDialog()`: Метод для запуска диалога с помощью Telegram API.

### Прочие компоненты
- Включаются вспомогательные методы для управления взаимодействиями с ботом Telegram, такие как отправка сообщений и обработка обновлений.
