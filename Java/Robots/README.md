## Учебный проект

Создание видеоигры


Консольное приложение для управления роботами.

## Описание
Программа предоставляет пользователю интерфейс для управления роботами. Включает в себя основное окно приложения и всплывающие окна для игры, просмотра координат роботов и лога.

## Использование
1. Запустите приложение `RobotsProgram`.
2. Используйте меню для изменения внешнего вида и проведения тестовых действий.
3. В главном окне приложения доступны игровое окно, окно с координатами роботов и окно просмотра лога.

## Требования
- Java Runtime Environment (JRE) версии 8 или выше.
